# web-habit.tracker

🌱 Habit Tracker

Consistency builds character.
A clean, minimal web app to track your daily habits, visualize streaks, and reflect on your monthly progress.

🔗 Live Demo: mahek-khan.github.io/web-habit.tracker

✨ Features

➕ Add Daily Habits — Pick an emoji and name your habit in seconds
📊 Stats Dashboard — See your Completion Rate, Best Streak, Done Today count at a glance
🔥 Streak Tracking — Stay motivated with your current and best streaks
📅 Monthly Progress — Visual progress bar showing how consistent you've been this month
🗺️ Heatmap View — GitHub-style weekly heatmap to spot your patterns
📈 Daily Completion Trend — Chart showing your consistency over time
📝 Monthly Thoughts — A built-in journal to reflect on your month

Pure HTML, CSS, JavaScript — no frameworks, no dependencies
LocalStorage for saving habits across sessions
Deployed via GitHub Pages


🚀 Getting Started
bash# Clone the repository
git clone https://github.com/mahek-khan/web-habit.tracker.git

# Navigate into the project
cd web-habit.tracker

# Open in browser
open index.html
Or just visit the live demo directly — no setup needed!

📖 How to Use

Pick an emoji that represents your habit (💪, 📚, 💧, etc.)
Type your habit name and click + Add
Check it off every day to build your streak
View your heatmap to see which days you were most consistent
Write your Monthly Thoughts to reflect on your progress


💡 What I Learned

Building a fully functional app with vanilla JavaScript (no libraries!)
Managing state with LocalStorage to persist data across browser sessions
Generating dynamic heatmap and chart visualizations from scratch
Deploying a static site with GitHub Pages
